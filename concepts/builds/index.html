Builds

A build is, minimally, an execution of a script in a container with a configured base image. If the script exits 0, the build succeeds. Otherwise, it fails.

A build can either be executed by a Job or executed manually with the Fly commandline. Both execute the same configuration, giving the guarantee that locally-executed builds with Fly are running the same way they would in CI.
